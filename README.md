## Features
- GODMODE
- all characters
- infinity air (fly)
- infinity lives
- infinity shields
- infinity boost
- x10 speed

## Controls
- press `1` to enable infinity air (fly)
- press `2` to enable infinity lives
- press `3` to enable infinity shields
- press `4` to enable infinity boost
- press `5` to enable x10 speed
- press `6` to enable god mode
